Step 1: Prepare Electronics

• Flash ESP32 with Sanctuary OS firmware

• Test all sensors on breadboard first

• Apply conformal coating to exposed traces

• Allow 24h cure time before assembly

Step 2: Wire Sensor Network

BME280 (I2C):
  VCC → ESP32 3.3V
  GND → ESP32 GND  
  SCL → GPIO 22
  SDA → GPIO 21

PIR Sensor:
  VCC → ESP32 5V
  GND → ESP32 GND
  OUT → GPIO 18

Step 3: Power System

• Connect 18650 to TP4056 charging module

• Wire buck/boost converter for 3.3V output

• Test charging with USB-C connection

• Verify 60+ hour runtime in deep sleep

Step 4: Enclosure Prep

• Drill 6mm hole for antenna (if needed)

• Cut 20x20mm window for light sensor

• Install PG7 cable gland for charging

• Apply IP65 sealant around openings

WiFi Connection Issues

• Check 2.4GHz network availability (ESP32 doesn't support 5GHz)

• Verify SSID/password configuration

• Test signal strength at installation location

• Check firewall settings for IoT devices

Sensor Reading Errors

BME280 returns -999:

• Check I2C wiring (SDA/SCL correct?)

• Verify 3.3V power supply stability

• Test with known-good sensor

Power Issues

Short battery life (<24 hours):

• Check deep sleep implementation

• Verify current draw <10mA in sleep

• Test charging circuit LED indicators

• Replace battery if capacity degraded
